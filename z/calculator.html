About This Page

This page demonstrates many core HTML tags and attributes:

  1. Structural tags: header, nav, main, section, article, aside, footer
  2. Form elements: input (text, number, radio, checkbox), select, fieldset, legend, label, button
  3. Table elements: table, thead, tbody, tfoot, tr, td, th, caption
  4. Text formatting: em, strong, code, br, hr

Tip: To make this calculator actually work, you'll eventually need JavaScript to read input values and compute the result. That's the natural next step after HTML!
