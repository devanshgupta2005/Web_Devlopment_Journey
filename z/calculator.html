Basic HTML Calculator

This is a pure HTML calculator layout. No CSS or JavaScript is used, so the buttons do not perform real calculations yet.

What Each Button Means

Button Meaning Needs JavaScript to Work?
0-9 Digit buttons for entering numbers Yes
. Decimal point, for entering fractional numbers Yes
+ - × ÷ Basic math operators: add, subtract, multiply, divide Yes
% Percentage / modulus operation Yes
= Calculates and shows the final result Yes
C Clears the display back to empty/zero Partially - form reset works without JS, but a true "clear display" needs JS
Backspace Removes the last entered digit Yes

Notes

In this project, every button is visually present but only form-related buttons (Reset and Submit) do anything at all without JavaScript. The digit and operator buttons currently do nothing when clicked because HTML alone cannot run logic or perform math.
